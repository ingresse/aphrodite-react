# Collapsible
### An alternative to accordion.

```jsx
import { Collapsible, Styled } from '@ingresse/aphrodite';
...
<Collapsible header="Hi">
    <Styled padding="5px 15px">
        <Text>
            there
        </Text>
    </Styled>
</Collapsible>
```
