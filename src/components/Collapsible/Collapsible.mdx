# Collapsible
#### An alternative to Accordion.

```jsx
import { Collapsible, Segment } from 'aphrodite-react';
...
<Collapsible header="Hi">
    <Segment padding="5px 10px">
        <Text>
            there
        </Text>
    </Segment>
</Collapsible>
```
