# Grid
#### Align the components with Aphrodite standards

```jsx
import { Container, Row, Col } from 'aphrodite-react';
...
<Container>
    <Row>
        <Col xs="12" sm="6">
            YOUR CONTENT HERE
        </Col>
    </Row>
</Container>
```
