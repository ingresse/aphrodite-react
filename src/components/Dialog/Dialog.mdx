# Dialog
#### Dialog boxes tell users about a task and can contain critical information, require decisions, or involve multiple tasks.

```jsx
import { Dialog } from 'aphrodite-react';

...

<Dialog
    opened={opened}
    onClose={() => setOpened(false)}
>
    <Button
        onClick={() => setOpened(false)}
    >
        Close
    </Button>
</Dialog>
```

## Examples
<Playground style={{ padding: '20px' }}>
    {() => {
        const [opened, setOpened] = useState(false);

        function handleDialog() {
            setOpened(!opened);
        }

        return (
            <Row>
                <Col xs={12}>
                    <Button onClick={handleDialog}>Open a simple Dialog</Button>
                    <Dialog
                        opened={opened}

                            onClick={() => setOpened(false)}
