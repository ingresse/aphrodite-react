# Clipboard
### A simple implementation of awesome library [React Clipboard.JS](https://github.com/nihey/react-clipboard.js), check it out there to more usage details.


```jsx
import { Clipboard } from 'aphrodite-react';

...

<Clipboard
    data-clipboard-text="Hello World">
    Hello
</Clipboard>
```

<Playground>
