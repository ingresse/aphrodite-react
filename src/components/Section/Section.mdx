# Section
### A section block with some predefinitions. "Styled" as base. "SectionTitle" as element title.

```jsx
import { Section } from '@ingresse/aphrodite';

<Section title="Hello">
    World
</Section>
```

## Advanced

        </Section>
        <Section
            loading
            padding="10px 15px"
            title="Loading state"
        >
            <ItemContent loading />

                The "loading" prop changes text color, if your app have some loading state case like this.
            </ItemContent>
            <ItemContent loading placeholderWidth={180} />
        </Section>
        <Section
            withLayer={false}
            title="A section without 'Layer' as content's wrapper"
        >
            Won't maded to use in this way, but can be necessary in some cases.
        </Section>
    </Styled>
