# Tooltip
#### Just an implementation of awesome library [**React Tooltip**](https://github.com/ReactTooltip/react-tooltip/tree/v4.2.21#react-tooltip), with Aphrodite's definitions.
#### So, it's just consult the [documentation](https://github.com/ReactTooltip/react-tooltip/tree/v4.2.21#react-tooltip) for more options.
```jsx
import { Tooltip } from '@ingresse/aphrodite';

<Tooltip tip="How you doing?!">Hover me</Tooltip>
```
