# Progress Bar
```jsx

<ProgressBar
    size="200px"    // Have priority
    percentual={60} // Will be used if "size" is inexistent
    animated={true}
    gradient={{ start: colors.get('tangerine'), end: colors.get('tangerine', 'dark') }}
    width={460}
/>
```
