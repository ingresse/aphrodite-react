# Styled
### A simple component block which accepts custom props and styles.

.

## "Responsive" using `styles` and `stylesXS` or `stylesGTXS`

<Playground>
    <Styled
        shadow
        shadowColor="secondary,light"
        radius={12}
        padding={16}
        margin="32px 0"
        textColor="white"
        styles={{
            // General Styles
            background: 'blue',
        }}
        stylesXS={{
            // Styles to screens less than SM (767px or less)
            background: 'orange',
        }}
    >
        "Styled" renderized with "section" tag instead of "div".
    </Styled>
    <Styled
        margin="32px 0"
        padding={16}
        radius={24}
        textColor="white"
        styles={{
            // General Styles
            background: 'lightgreen',
        }}
        stylesGTXS={{
            // Styles to screens greater than XSM (476px or less)
            background: 'gray',
        }}
    >
        "Styled" renderized with "section" tag instead of "div".
    </Styled>
</Playground>
