# Form | Input Date (Fallback)
### A simple alternative to manage input date. It's used as fallback of DatePicker component.

This component works with [Day.js](https://day.js.org/) to validations and [react-input-mask](https://www.npmjs.com/package/react-input-mask) to mask pattern.

## Examples
<Playground>
    {() => {
        const [example, setExample] = useState('');

        return (
            <Row>
                <Col as={Styled} xs={12} sm={6} padding="10px">
                    <div>
                        Regular InputDateFallback
                    </div>
                    <br />
                    <InputDateFallback
                        id="example"
                        value={example}
                        onChange={({ target }) => { setExample(target.value); console.log('InputDateFallback target', target) }}
                        label="Type a date"
                    />
                    <br />
                    <div>
                        <strong>input value: </strong>
                        {JSON.stringify(example)}
                    </div>
                </Col>
            </Row>
        );
    }}
</Playground>
