# Lists
#### Organizing information with an alternative to tables.

```jsx
import { ListGroup, ListHeader, ListItem } from 'aphrodite-react';
...
<ListGroup>
    <ListHeader>
        Title
    </ListHeader>
    <ListItem>
        Item One
    </ListItem>
    <ListItem>
        Item Two
    </ListItem>
</ListGroup>
```
