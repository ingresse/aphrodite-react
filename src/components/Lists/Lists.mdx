# Lists
#### Organizing information with an alternative to tables.

```jsx
import { Input, ListGroup, ListItem } from 'aphrodite-react';
...
<ListGroup
    radius={window.innerWidth < 768 ? 0 : 4}
    noSideBorders={window.innerWidth < 768}>
    <div>
        <ListItem header>
            <Input type="checkbox" id="firstToggle" labelRight={window.innerWidth < 768}>
                Header
            </Input>
        </ListItem>
        <ListItem>
            Item ...
        </ListItem>
    </div>
    <div>
        <ListItem header>
            <Input type="checkbox" id="secondToggle" color="#ff0000">
                Header
            </Input>
        </ListItem>
        <ListItem>
            Item ...
        </ListItem>
    </div>
</ListGroup>
```
