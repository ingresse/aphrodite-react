# Lists
#### Organizing information with an alternative to tables.

```jsx

...

    <List>
        <ListItem>
            Ingresse
        </ListItem>
        <ListItem>
            Aphrodite
        </ListItem>
        <ListItem>
            React
        </ListItem>
    </List>
);
```
