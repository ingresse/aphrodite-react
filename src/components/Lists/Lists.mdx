# Lists
#### Organizing information with an alternative to tables.

```jsx
import { List, ListItem } from 'aphrodite-react';

...

    <List>
        <ListItem>
            Ingresse
        </ListItem>
        <ListItem>
            Aphrodite
        </ListItem>
        <ListItem hoverable>
            React
        </ListItem>
    </List>
);
```
