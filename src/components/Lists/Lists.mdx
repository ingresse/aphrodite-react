# Lists
### Organizing information with an alternative to tables.

```jsx
import { List, ListItem } from '@ingresse/aphrodite';

...

return (
    <List>
        <ListItem>
            Ingresse
        </ListItem>
        <ListItem>
            Aphrodite
        </ListItem>
        <ListItem padding="15px 40px">
            React
        </ListItem>
        <ListItem hoverable>
            UI
        </ListItem>
        <ListItem withLink>
            <a target="_blank" href="https://ingresse.github.io/aphrodite-react">
                Facilities
            </a>
        </ListItem>
    </List>
);
```
