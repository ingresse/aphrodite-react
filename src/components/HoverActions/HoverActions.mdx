# HoverActions
### On some content hovered, display another content over the original.

```jsx
import { HoverActions, HoverActionsItem } from 'aphrodite-react';

...

<HoverActions
    actions={(
        <Fragment>
            <HoverActionsItem>
                Useless
            </HoverActionsItem>
            <HoverActionsItem>
                Useless
            </HoverActionsItem>
        </Fragment>
    )}>
    My Hoverable Element expecting some surprising actions
</HoverActions>
```
