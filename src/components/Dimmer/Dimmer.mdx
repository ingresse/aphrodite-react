# Dimmer
### Appear over everything of parent element. Inspired by [Semantic UI's Dimmer](https://semantic-ui.com/modules/dimmer.html).


## Example

<Playground>
    {() => {
        const [dimmer, setDimmer] = useState(true);

        return (
            <Card padding="15px 20px">
                <div>
                    Mussum Ipsum, cacilds vidis litro abertis. Mauris nec dolor in eros commodo tempor. Aenean aliquam molestie leo, vitae iaculis nisl. Não sou faixa preta cumpadi, sou preto inteiris, inteiris. Leite de capivaris, leite de mula manquis sem cabeça. Sapien in monti palavris qui num significa nadis i pareci latim.
                </div>
                <div>
                    Casamentiss faiz malandris se pirulitá. Admodum accumsan disputationi eu sit. Vide electram sadipscing et per. Interessantiss quisso pudia ce receita de bolis, mais bolis eu num gostis. Cevadis im ampola pa arma uma pindureta.
                </div>
                <Styled padding="15px 0 5px">
                    <Button
                        small
                        disabled={dimmer}
                        onClick={() => setDimmer(true)}
                    >
                        Open Dimmer
                    </Button>
                </Styled>

                <Dimmer
                    id="dimmer_example_one"
                    opened={dimmer}
                    openedCallback={setDimmer}
                    icon={{ slug: 'flash-off' }}
                    title="Dimmer Example"
                >
                    Content here
                </Dimmer>
            </Card>
        );
    }}
</Playground>
