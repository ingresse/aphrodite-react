# Utilities | Gradients
### Tools to made easy the development with our gradients presets.

## Usage
#### The `gradients` utility will help you to get the desired color from Aphrodite React's Gradients Palette.
```
import { gradients } from '@ingresse/aphrodite';

gradients.palette; // All availables gradients palette.
gradients.keys; // All availables gradients palette keys.
gradients.list; // All availables gradients palette values.

gradients.random();         // Get a random gradient start/end object.
gradients.get('tangerine'); // Get specific gradient.

gradients.backgroundLinear('tangerine', 45); // CSS string to apply as background.
```
