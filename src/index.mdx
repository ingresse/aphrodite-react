# Aphrodite
### React UI Library from [ingresse.com](https://www.ingresse.com)

---

## Install
#### You can install with Yarn or NPM in your App Project, but using GitHub repository address as source:
```
yarn add https://github.com/ingresse/aphrodite-react.git
```

---

## Usage Example
#### Include Aphrodite Components inside your React application.

`src/components/index.js`:
```js
export {
    Button,
    Container,
    Segment,
} from 'aphrodite-react';
```

`src/App.jsx`:
```jsx
/* Core Packages */
import React from 'react';
import propTypes from 'prop-types';

/* Components */
import {
    Button,
    Container,
    Segment,
} from './components';

/* Component Itself */
function App() {
    function handleClick(evt) {
        console.log('button click', evt);
    }

    return (
        <Container>
            <Segment
                textAlign="center">
                <Button onClick={handleClick}>
                    Aphrodite Button
                </Button>
            </Segment>
        </Container>
    );
}

/* Exporting */
export default App;
```
