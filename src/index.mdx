# Aphrodite
### React UI Library from [ingresse.com](https://www.ingresse.com)

---

## Install
#### You can install with Yarn or NPM in your App Project:
```
npm i --save @ingresse/aphrodite
```

---

## Usage Example
#### Re-export the Aphrodite tools through your React application folder structure.

`src/utils/index.js`:
```js
export {
    colors,
} from 'aphrodite-react';
```

`src/components/index.js`:
```js
export {
    Button,
    Container,
    Segment,
} from 'aphrodite-react';
```

#### Then, it's easy to use
`src/App.jsx`:
```jsx
/* Core Packages */
import React from 'react';

/* Components */
import {
    Button,
    Container,
    Segment,
} from './components';

/* Component Itself */
function App() {
    function handleClick(evt) {
        console.log('button click', evt);
    }

    return (
        <Container>
            <Segment
                textAlign="center">
                <Button onClick={handleClick}>
                    Aphrodite Button
                </Button>
            </Segment>
        </Container>
    );
}

/* Exporting */
export default App;
```
